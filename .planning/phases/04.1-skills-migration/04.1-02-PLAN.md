<objective>
Rewrite the Claude Code adapter to install 8 per-command Skills into `.claude/skills/codewiki-<name>/SKILL.md` instead of 8 flat slash-command files into `.claude/commands/codewiki/`. Update the instructions template, integration tests, and pack test to match the new install surface per D-01, D-02.

Purpose: The adapter is the bridge between template source files and the user's project. This plan makes it write skills (one directory per command) instead of flat command files, and verifies the new layout end-to-end through integration and pack tests.

Output: Working `npx codewiki init --tool claude-code` that installs skills to `.claude/skills/codewiki-<name>/SKILL.md`, agents to `.claude/agents/`, hooks to `.codewiki/hooks/`, and settings/instructions as before.

**src/lib/merge.ts** — Add `deduplicateHookEntries` function for object-array deduplication (needed for settings.json hooks which are arrays of objects, not strings). Per research assumption A2, compare by `JSON.stringify` of each entry for deduplication:
```typescript
export function deduplicateHookEntries<T>(entries: T[]): T[] {
  const seen = new Set<string>();
  return entries.filter(entry => {
    const key = JSON.stringify(entry);
    if (seen.has(key)) return false;
    seen.add(key);
    return true;
  });
}
```

Also export `deduplicateHookEntries` from `src/lib/index.ts`.
